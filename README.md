# Board Pen

A web-based digital pen that uses your computer camera to track hand gestures for drawing on a virtual canvas.

## Gestures

- **1 Finger (Index Finger):** Draw on the canvas.
- **2 Fingers (Index + Middle):** Erase lines by hovering over them.
- **Spread Hand (All Fingers):** Move the cursor without drawing.
- **Spacebar:** Clear the entire canvas.

## Setup

Serve the files using a local development server (like VS Code Live Server) so that camera permissions can be properly granted by the browser.
